| 方法 | 状态 | 说明 |

| `greedy_stdma_2hop` | 已实现 | 严格两跳安全 STDMA 贪心近似，只申请上一帧观测中无一跳/两跳占用风险的 slot，确定性申请 |

## 论文口径

推荐主表方法：

```text
plain_tdma heuristic_only greedy_stdma_2hop traffic_adaptive_tdma ppo_baseline B
```

推荐附录方法：

```text
greedy_stdma zmac_inspired trama_inspired bandit_index_proxy
```

写作约束：

- `greedy_stdma_2hop` 可写成 strict two-hop-safe greedy STDMA approximation。
- `zmac_inspired` / `trama_inspired` 只能写 inspired baseline。
- 不要写“完整复现并超过 Z-MAC/TRAMA/STDMA 原论文协议”。
